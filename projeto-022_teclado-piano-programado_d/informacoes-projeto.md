# Yarn - Ambiente Virtual Node?

Link como instalar Yarn:
https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable

Instalar Gerenciador de Pacotes:
=> npm install --global yarn

Verificar Versão:
=> yarn --version

Entrar na Pasta Específica com cd ...

=> yarn init -y
